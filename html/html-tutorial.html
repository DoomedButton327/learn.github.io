HTML Commands and Explanations

HTML Basics

HTML stands for HyperText Markup Language. It is used to structure content on the web. Example tags include <html>, <head>, and <body>.

HTML Tags

HTML tags define elements on a webpage. Examples include <div>, <p>, <h1> - <h6>, and <a>. Each tag has a start and end tag.

HTML Attributes

Attributes provide additional information about HTML elements. Common attributes include 'href' for links, 'src' for images, and 'class' for CSS styling.
